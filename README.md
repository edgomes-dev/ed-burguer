## Tecnologias Utilizadas

- Frontend

  - Typescript
- Backend

  - Spring Boot
  - Spring Data
  - Spring Security
  - JUnit5
  - Docker
  - Postgres

- Back está pronto praticamente, muito bem estruturado com arquitetura de camadas bem definidas no padrão API Rest.
